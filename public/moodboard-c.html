Style C Key Principles

  • — Restraint：最多 2 色（白 + Teal）
  • — Type-first：字型層次取代裝飾
  • — Whitespace：留白即設計
  • — SVG icons：線條簡潔，無 emoji

Differentiators vs Style A/B

  • — 無漸層背景，純白為主
  • — 卡片 hover = 左側色條 + shadow
  • — Feature Alternating（非純 grid）
  • — 業師列表 vs 業師卡片

Font Stack

  • — Primary：Plus Jakarta Sans
  • — Chinese：Noto Sans TC
  • — Quote only：Noto Serif TC italic
  • — Weights：400 / 600 / 700 / 800
